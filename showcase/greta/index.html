The course

  • Theory
    • Overview of Natural Language Processing
      • Principles of tokenisation, lemmatisation , Part-of-Speech tagging, morphological analysis, named entity recognition, syntactic parsing, semantic analysis, etc.
  • Practice
    • Text preparation with the command line: count, sort, extract, format, clean data
    • Lemmatisation and PoS-tagging (with TreeTagger): basics of any NLP project

  • Not covered
    • Programming
